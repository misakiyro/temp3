## 使用指南

### 安装

```bash
# 克隆仓库
git clone https://github.com/misakiyro/temp3.git

# 进入项目目录
cd temp3
```

### 配置

1. 复制配置文件模板
2. 2. 根据需要修改配置参数
   3. 3. 保存配置文件
     
      4. ### 运行
     
      5. ```bash
         # 运行项目
         npm start
         ```

         ### 常见问题

         - 如遇到依赖问题，请先执行 `npm install`
         - - 确保Node.js版本 >= 14.0.0
