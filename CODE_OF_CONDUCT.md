Pledge  
We pledge to foster a welcoming and inclusive environment for all contributors to our open source project. We are committed to upholding high standards of conduct and treating everyone with respect and dignity.  

Standards  
All participants in this project are expected to adhere to the following standards:  
- Communicate respectfully and avoid harassment, discrimination, or offensive language.
- - Maintain professionalism in all interactions, including code reviews, discussions, and documentation.
  - - Uphold transparency and honesty in contributions and reporting issues.
   
    - Responsibilities
    - Each contributor is responsible for:
    - - Ensuring their actions align with the project's Code of Conduct.
      - - Reporting any violations of this code to the project maintainers.
        - - Supporting a positive and inclusive community by addressing conflicts constructively.
         
          - Scope
          - This Code of Conduct applies to all project-related activities, including but not limited to:
          - - Code contributions, pull requests, and issue reporting.
            - - Communication channels such as forums, chat rooms, and email.
              - - Public events, workshops, and presentations associated with the project.
               
                - Enforcement
                - Violations of this Code of Conduct will be addressed by the project maintainers. Depending on the severity, actions may include:
                - - Private warnings or requests for clarification.
                  - - Public reprimands or temporary suspensions.
                    - - Permanent removal from the project community.
                      - Maintainers reserve the right to interpret and apply these rules fairly and consistently.
